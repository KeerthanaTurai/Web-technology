Course Details

Course Name : WEB TECHNOLOGIES
YEAR/SEM : III/I

   Course Outcomes

   Students will be able to


  1. Develop static web pages using HTML tags (L3)



  2. Apply style sheets to web pages with data validation (L3)



  3. Create Machine understandable web documents usingXML (L6)



  4. Develop servlet based dynamic web application (L3)



  5. Design data driven web application (L6)


    6.

UNIT - I

   Introduction to Web:

   Understanding Internet and Web, Web Architechture, Web servers, protocols: HTTP


   Introduction to HTML:

   History of HTML, www, HTML Basics : Elements, Attributes, Tags, Tables, Forms, Frames.div and span tags


   Course Outcome:

   Develop static web page using HTML tags

UNIT - II

   CSS:

   Introduction to Cascading Style Sheet, Types of style sheets, page layout, selectors, pseudo classes and elements


   JAVA SCRIPT:

   Introduction to scripting, control structures, conditional statements, Array functions, objects


   HTML DOM:

   Predefined object (Window, Location, History, Navigator), Events, DOM Node methods, Navigation
   Nodes: creating nodes, adding nodes, inserting nodes, removing & replacing nodes,
   Form object and elements, DHTML with Java script



   Course Outcome:

   Apply style sheets to web pages with datac validation

UNIT - III

   XML:

   Basic of XML, Elements, Attributes, validation, Nameapace
   XML Scheme Languages: Introduction to DTD, internal and external DTD, Elements of DTD
   DTD Limitations, XML Schema, Schema structure, Elements, parsing using XML


   XML DOM:

   Document node, element node, Text node, Java and DOM, Navigating DOM Tree


   Course Outcome:

   Create Machine understandable web documents using XML




<

UNIT - IV

   AJAX:

   Introduction, Environment, Asynchronous communication, process steps, sending and retrieving information, Ajax with XML


   Servlets:

   Introduction, Lifecycle, Generic and HTTP servlet, passing parameters to servlet, HTTP servlet Request & Response interfaces
   Deploying web applications
    Session Tracking: Hidden form fields, cookies, URL-Rewriting, session


   Course Outcome:

   Develop servlet based dynamic web application

UNIT - V

   JSP:

   Introduction, Difference between servlets & JSP, Anatomy of JSP page,
   JSP elements: Directives, comments, expressions, script lets, Declaration,
   Implicit JSP objects, using Action elements


   JDBC:

   Introduction, JDBC Drivers, Loading Driver, establishing connection,
   Executing SQL statement in JSP pages, MVC Architechture


   Course Outcome:

   Design data driven web applications
