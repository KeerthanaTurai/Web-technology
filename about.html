Web Technologies

Web Development is a digitial art which help people to reach their customers in an innovative manner. In simple it is the creation and maintainance of websites. Web application development is the creation of application programs that reside on remote servers and are delivered to the user's device over the internet. A web application also called as webapp does not need to be downloaded and is instead accessed through a network. An end user can access a web application through a web browser such as Google chrome, Safari, or Mozilla FireFox. Many of the web applications are written in Java Script, Cascading Style Sheets(CSS) and HTML. Mostly web develpoers use languages like HTML, CSS, Javascript, XML and many more.
